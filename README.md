# Question And Answers Api
MVP Restful API written in Go


Basic Requirements: 
 - MongoDB and Docker

Setting up the service:
1 - It is necessary Compose Up docker-compose.yml file, in order to generate the right mongoDB configuration (image, containers and network)
2 - Another solution is manually create a mongoDb server, just reading the  docker-compose.yml file
3 - Running the service is simple, just type 'go run main.go' or build it typing: 'go build main.go'.
4 - All the instructions to use this service is automatically generated by swagger in http://localhost:3000/swagger/index.html
5 - Missing, could be improved to the next releases:
	Unit tests
	Run the application in docker container
	Prepare the application to be managed by kubernets with docker
	Caching
	Resiliency
